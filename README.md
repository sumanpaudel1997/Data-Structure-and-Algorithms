## Data Structures

### Stack

A stack is a Last-In-First-Out (LIFO) data structure. It supports two main operations: `push` (add an item) and `pop` (remove the top item).

### Queue

A queue is a First-In-First-Out (FIFO) data structure. It supports `enqueue` (add an item) and `dequeue` (remove the front item) operations.

### Linked List

A linked list consists of nodes, where each node points to the next node. It can be singly linked or doubly linked.
